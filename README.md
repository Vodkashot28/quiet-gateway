# quiet-gateway

A resurrection gateway for hosting IPFS artifacts, IPNS echoes, DNSLink diagnostics, and Getgems domain badges.

## Routes
- `/` → Hello world
- `/quiet/` → Quiet Reference site
- `/cid/<CID>/` → Artifact lineage
- `/ipns/<PeerID>/` → IPNS echoes
- `/dnslink-check` → DNSLink validator
- `/car/<name>.car` → CAR file hosting
- `/ledger/` → Resurrection logs
